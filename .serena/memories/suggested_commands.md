Root: `pnpm dev` starts frontend+backend, `pnpm dev:all` starts frontend+backend+worker, `pnpm build` builds shared/backend/frontend, `pnpm lint` runs workspace lint, `pnpm typecheck` runs workspace typecheck. Backend: `pnpm --filter @sui-portfolio/backend dev`, `pnpm --filter @sui-portfolio/backend build`, `pnpm --filter @sui-portfolio/backend seed:dev`. Frontend: `pnpm --filter @sui-portfolio/frontend dev`. Worker: `pnpm --filter @sui-portfolio/worker dev` if added later.
